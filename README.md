### 2. Shadowrocket (小火箭) 无法测试延迟？

请进入小火箭设置，将“延迟检测方式”从 TCP 调整为 **ICMP**，即可正常查看节点状态。

### 3. 鸿蒙纯血系统 (HarmonyOS Next) 如何安装？

1. 在官方应用商店安装 **“出境易”** 或 **“卓易通”**。
2. 在该环境下安装安卓版本的代理 APP（如 Sing-box）。
3. 参考安卓教程完成配置。

### 4. 节点全部超时/Error 怎么办？

* **DNS 覆写**：FlClash 用户请在 `设置 -> 工具 -> 覆写 -> DNS` 中勾选“覆写DNS”并重启。
* **模式切换**：尝试打开 Clash 的 **Tun 模式**。
* **网络环境**：尝试切换 Wi-Fi 或使用手机热点。
